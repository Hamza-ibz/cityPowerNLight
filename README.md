# Case Management System

This project demonstrates the use of a case management system where you can create, read, update, and delete cases (incidents) and associated contacts. The system is built using C# and follows a simple architecture that includes controllers for managing cases and contacts, along with services for interacting with a backend database.

## Features

- **Create** a new Account, Contact and Cases.
- **Read** all Account, Contact, Cases and display them in a formatted table.
- **Update** a Account, Contact and Cases with new information.
- **Delete** a Account, Contact and Cases.
- **Error handling** for each operation to ensure smooth execution.

## Prerequisites

- .NET SDK 6.0 or later
- Visual Studio or any C# IDE (such as JetBrains Rider)
- A database or mock data service for storing incidents and contacts
